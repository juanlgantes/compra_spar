🛒 Mi Pedido Spar AppMi Pedido Spar es una aplicación web ultra-optimizada y de uso personal diseñada bajo una estricta filosofía de Fricción Cero. Su objetivo principal es reducir al mínimo la carga cognitiva del usuario al gestionar, calcular y enviar su lista de la compra habitual al supermercado.Nacida de la consolidación de decenas de archivos Excel históricos y tickets físicos, la aplicación ha evolucionado hasta convertirse en una central de compras inteligente sincronizada en la nube.✨ Características Principales (Superpoderes)🧠 Base de Datos Histórica: Inventario nativo creado a partir del escaneo de 53 iteraciones de compras anteriores. Los precios base fueron extraídos y promediados mediante OCR de tickets físicos.☁️ Sincronización Familiar (Espejo): Conexión en tiempo real entre múltiples dispositivos (móvil y PC) a través de Firebase Firestore usando un código unificado de hogar.🏷️ Motor Financiero Multi-Nivel: Cálculo dinámico de gasto con alertas en vivo para alcanzar metas de cupones de descuento (Ej: -6€ al gastar 60€, -12€ al gastar 100€).📦 Sistema de Lotes (Batching): Agrupación de compras repetitivas (Ej: "Compra de Fin de Mes") en paneles desplegables interactivos para edición rápida y volcado a un clic.⚡ Generación de Envíos Automatizada: Compilación y apertura automática del cliente de correo con el pedido estructurado por pasillos, notas personalizadas y la tarjeta de socio inyectada.📋 Exportación Ofimática: Capacidad de copiar el pedido en formato tabla (TSV) compatible de forma nativa con Microsoft Excel o Google Sheets.🛠️ Arquitectura TécnicaEl proyecto opera bajo un Paradigma Single-File (Archivo Único) para garantizar la máxima portabilidad y simplicidad de despliegue.Frontend: Desarrollado en React (JSX) para permitir un renderizado reactivo sin recargas, con estilización de interfaz a través de Tailwind CSS (Mobile-first).Backend / Nube: Integración ligera con Google Firebase (App, Auth, Firestore). Utiliza autenticación anónima/por tokens y colecciones públicas seguras para el mirroring en tiempo real.Almacenamiento Local: Uso intensivo de localStorage para persistir configuración de cupones, tarjeta de cliente y el identificador de la familia.🚀 Instalación y DespliegueDada su arquitectura consolidada, no requiere procesos de build complejos ni servidores de aplicaciones pesados.Abre el archivo principal de la aplicación (pedido_spar_nube.jsx o su versión compilada en HTML).Sírvelo en cualquier servidor web estático local (Ej: python3 -m http.server 8000) o súbelo a un hosting web estático (GitHub Pages, Vercel, o tu servidor OVH).Para vincular un nuevo dispositivo:Ve a la rueda de Configuración (⚙️).Obtén el Código Compartido Familiar del dispositivo original.Introdúcelo en el nuevo dispositivo y pulsa "Vincular". La sincronización será inmediata.🗺️ Estructura del Repositorio y GobernanzaEste proyecto está gobernado por directivas estrictas de inteligencia artificial y desarrollo continuo.
